Expedition {Экспедиция}

Expedition_Info 
{
Проведение экспедиции позволит начать постройку таких строений, как плантация и рудники. Возможность провести экспедицию появляется после расширения колонии.
}


StoreHouse {Склад}

StoreHouse_Info 
{
На складе вы сможете оставить на хранение любые товары. Кроме того, склад можно расширить для большей вместительности. Расширение склада доступно после расширения колонии.
}

Plantation {Плантация}

Plantation_Info 
{
Плантация позволяет добывать сахар и кофе. Чтобы начать её постройку необходимо сперва произвести эскпедицию. Кроме того, имеется возможность расширить плантацию для большего количества добываемого сахара, рабов и надсмотрщиков. Именно после расширения доступна добыча кофе.
}

GoldMine {Золотой рудник}

GoldMine_Info 
{
Постройка данного рудника позволит добывать золотую руду наряду с серебряной и железной. Доступен к постройке после проведения экспедиции. Кроме того, имеется возможность расширить рудник для большего количества добываемой руды, рабов и надсмотрщиков.
}
